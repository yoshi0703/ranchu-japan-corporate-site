ホーム / サービス / ホームページ制作

「ホームページはあるけど、問い合わせが来ない。」

その原因は、デザインではなく情報設計にあるかもしれません。
Ranchu Japanは、検索にもAIにも正しく届く構造で、質の良いお客さまを呼ぶサイトをつくります。

SHIFT

ポストAI時代、ホームページに求められること

ChatGPTやGeminiが検索結果を要約する時代。ユーザーはGoogle検索だけでなく、AIに質問して情報を得るようになっています。

この変化に対応するには、従来のSEO（検索エンジン最適化）に加えて、AEO（Answer Engine Optimization） — AIが理解しやすい構造化された情報設計が必要です。

APPROACH

私たちのつくり方

Step 1

情報設計から始める

「誰に、何を、どの順番で伝えるか」を設計します。キーワードだけでなく、ユーザーの検索意図と行動導線が設計の起点です。

Step 2

SEO/AEO最適化

セマンティックHTML、構造化データ(JSON-LD)、FAQ構造化。人間にもAIにも正しく理解される技術基盤を実装します。

Step 3

成果に直結する導線

問い合わせ・来店・購入など、ビジネスゴールに最短で到達する導線設計。見た目の美しさだけでなく、成果を出すサイトに。

FEATURES

標準で含まれること

技術設計

セマンティックHTML、構造化データ(JSON-LD)、Core Web Vitals最適化、完全レスポンシブ設計。表示速度にもこだわります。

コンテンツ設計

ユーザーインテント分析、FAQ構造化、ローカルSEO対応、AEO向けコンテンツ構造。「本当に知りたい情報」を届けます。

DIFFERENCE

一般的なHP制作会社との違い

テンプレートにテキストを流し込むだけの制作では、他社との差別化ができません。Ranchu Japanは、あなたのビジネスの強みを言語化し、それが正しく伝わる情報構造を設計します。

このサイト自体が、私たちの制作物です。構造化データ、セマンティックHTML、Core Web Vitals最適化 — すべてこのサイトで実践しています。

FAQ

よくあるご質問

制作期間と費用の目安は？

5〜10ページ規模のサイトで1〜2ヶ月、50万円〜が目安です。ページ数や機能によって変動するため、まずはヒアリングで概算をお伝えします。

WordPressで作りますか？

目的に応じて最適な技術を選定します。更新頻度が高い場合はCMS、パフォーマンス重視の場合は静的サイトなど、運用のしやすさを第一に考えます。

既存サイトのリニューアルも対応できますか？

対応しています。現サイトのSEO資産を活かしながら、情報設計と技術基盤を刷新します。リダイレクト設計も含めて対応します。

あなたのビジネスに合うサイト、一緒に考えませんか？

初回ヒアリングは無料です。現状の課題整理からお手伝いします。
